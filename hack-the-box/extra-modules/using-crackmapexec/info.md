# Info

## Supported Protocols

| **Protocol** | **Default Port** |
| ------------ | ---------------- |
| SMB          | 445              |
| WINRM        | 5985/5986        |
| MSSQL        | 1433             |
| LDAP         | 389              |
| SSH          | 22               |
| RDP          | 3389             |
| FTP          | 21               |

## View a protocol's help menu

```
crackmapexec <protocol> --help
```

```
crackmapexec smb --help
```

## Protocol Modules

```
crackmapexec <protocol> -L
```

```
crackmapexec smb -L
```
